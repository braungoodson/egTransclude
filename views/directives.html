Directives

A directive is a powerful interface to the DOM. It can bind, transclude, replace, and much more.

Transclusion

In this example, we transclude a bootstrap media object with an angular directive, by wrapping the object in a bootstrap panel.

Simple Transclude

With ng-simple-transclude, a .media-object is wrapped in a .panel.panel-basic.
